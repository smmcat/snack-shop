# 小卖部商城

配合 Nodejs 后端项目 [YXstore-backend](https://github.com/smmcat/YXstore-backend) 进行的前端开发；

![img](https://github.com/smmcat/snack-shop/blob/main/src/assets/test.png)

该项目皆在于方便学生在小卖部购物和清单记录等功能。

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```
